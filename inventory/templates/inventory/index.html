 SignTyper for FSL

A web application for documenting signs in Filipino Sign Language (FSL)

Updates

  • Recognizable handshapes in Filipino Sign Language (FSL) are now in the database.
  • Entries for FSL corpora to be uploaded soon.

 SignTyper for FSL uses a multi-media approach for documenting FSL signs. Alongside sign gloss, it uses the Hamburg Notation System (HamNoSys) to describe individual signs. HamNoSys transcriptions are then converted into Sign Gestural Markup Language (SiGML). Through SiGML, it is able to synthesize natural sign language performance using virtual human characters using the JASigning system.
