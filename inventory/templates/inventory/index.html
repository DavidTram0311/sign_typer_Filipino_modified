 SignTyper

A web application for documenting signs in Filipino Sign Language (FSL)

 SignTyper uses a multi-media approach for documenting FSL signs. Alongside sign gloss, it uses the Hamburg Notation System (HamNoSys) to describe individual signs. HamNoSys transcriptions are then converted into Sign Gestural Markup Language (SiGML). Through SiGML, it is able to synthesize natural sign language performance using virtual human characters using the JASigning system. SignTyper offers a tool called SyntheSign which provides a more intuitive way to transcribe signs in HamNoSys and automatically convert HamNoSys to synthetic sign animation.
