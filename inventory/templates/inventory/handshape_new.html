HamNoSys Handshape Transcription Guide
  • This tool divides HamNoSys transcription for handshapes into two steps: base form and finger specifications.
  • The base form is defined by: basic hand shape, thumb modification, and bending.
  • Finger specifications are optional and are added only when changes are needed for the base form.
  • There are two separate textboxes that you can edit at each transcription step.
